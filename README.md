# Build An AI Voice Assistant Using Python

## This is an AI Assistant like Iron Man's Jarvis
## In this program I implemented an Friday voice assistant it can do the followings

 - Greetings
 - Taking Commands
 - Search in Chrome
 - Search in Wikipedia
 - Playing Songs
 - Sending Mails
 - Take a Screenshot
 - Remembering
 - Logout,shutdown,Restart
 - Make Jokes
